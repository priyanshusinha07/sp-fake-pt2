üß† What Is an API?

An API (Application Programming Interface) allows two systems to communicate. In fake website detection, the frontend sends a URL to the backend API, which then analyzes the URL and returns whether it‚Äôs fake or real.

üö¶ How It Works

  1. User enters a URL in the web app.
  2. Frontend sends a POST request to the API with the URL.
  3. Backend processes the URL using ML or rule-based logic.
  4. API sends back a JSON response with prediction and confidence.
Example Request: 
{
  "url": "https://scam-example.com"
}
Example Response: 
{
  "prediction": "fake",
  "confidence": 0.89
}

üõ† Technologies Used

  • HTML/CSS/JS: For building the user interface
  • JavaScript Fetch API: For sending API requests
  • Machine Learning Model (Backend): For URL classification
  • JSON: For request/response data exchange

üì¶ Sample JavaScript Code

Client-side API call using Fetch:

fetch("https://your-api.com/api/detect", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({ url: "https://example.com" })
})
.then(res => res.json())
.then(data => {
  console.log("Prediction:", data.prediction);
});

ü§î Why Detect Fake URLs?

Fake websites are used for phishing, data theft, and scams. Detecting them protects users from malicious attacks and fraud.
